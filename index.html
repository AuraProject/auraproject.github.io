A Location-based Chat Engine

...to communicate people around you anonymously

The user sees the area in his/her mobile phone as a 3D virtual chat room. Users are placed on a virtual ground with an icon and their auras. The user can change his/her auracolor, can type a message and send others some special kind of requests like mini games.
